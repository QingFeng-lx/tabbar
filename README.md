# tabbar
封装TabBar组件

## 技术栈
- vue 3
- vue-router
- webpack打包工具
## 项目构建
``` bash

# 安装依赖
npm install

# 启动服务 localhost:8080
npm run serve

# 构建生产环境
npm run build:prod
```
